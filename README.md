# Weather Dashboard App

A simple, responsive weather dashboard application that displays current weather conditions and forecasts for any city.

## Features

- 🔍 City search functionality
- 🌡️ Current weather conditions
- 💨 Wind speed and humidity display
- ⛅ Weather icons and conditions
- 📱 Responsive design with Tailwind CSS
- 🎨 Clean, modern UI

## Setup

1. Clone the repository:
```bash
git clone https://github.com/Rahulkumawat0/weather-app.git
cd weather-app
```

2. Get an API key from [OpenWeatherMap](https://openweathermap.org/api)

3. Update `config.js` with your API key:
```javascript
const CONFIG = {
  API_KEY: "your_api_key_here"
};
```

4. Open `index.html` in your browser and start searching for cities!

## Deployment

This app is deployed on Vercel and can be accessed at: https://weather-app-rahulkumawat0.vercel.app

## Technologies Used

- HTML5
- CSS (Tailwind CSS)
- JavaScript (Vanilla)
- OpenWeatherMap API
